# tslint-config

Just my config for [tslint](https://github.com/palantir/tslint)

## Installation

```
npm install --save-dev @danila/tslint-config
```

## Usage

### Just extend your `tslint.json`

```json
{
    "extends": "@danila/tslint-config",
    "rules": {
        // ...
        // Your rules
        // ...
    }
}
```
